## Installation

Ensure you have [pnpm](https://pnpm.io/) and Node.js v22+ installed.

1. **Clone the repository.**
2. **Install all dependencies** across workspaces:
   ```bash
   task install
   ```

## Running the Project

Start the local development server for both frontend and backend automatically:

```bash
task dev
```

See the `docs` directory for system design analysis and auditing details.
